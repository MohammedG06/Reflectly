# Reflectly

Reflectly is a SwiftUI-based journaling application developed for the **Apple Swift Student Challenge**.

The app provides dynamic journaling prompts that automatically adjust based on user mood and engagement patterns to encourage reflection and consistent journaling.

---

## Features

- Auto-adjusting journaling prompts
- Dynamic SwiftUI interface
- Personalized reflection prompts
- Simple and responsive user experience

---

## Technologies Used

- Swift
- SwiftUI
- Swift Playgrounds

---

## Project Background

This project was created as part of the **Apple Swift Student Challenge**, an international student coding competition hosted by Apple.

The goal of the project was to design a journaling tool that helps users reflect more consistently by adapting prompts based on their responses and engagement.

---

## How to Run the Project

1. Download the repository
2. Open the `.swiftpm` file
3. Launch the project using **Swift Playgrounds** or **Xcode**
4. Run the app to interact with the journaling interface
